\subsection{Detailed Description}
The declaration of major parsing functions used e.g for (re)implementation of new extended properties see class Property These function also could be used in a different, much wider contex. all of them have the semantic: {\ttfamily int parse ( TYPE\& result , const \hyperlink{classstd_1_1string}{std::string}\& input )} where {\ttfamily input} is the input string to be parsed, and {\ttfamily result} is the the result of parsing


\begin{DoxyCode}
  const std::string        input = ... ;
  std::vector<std::string> result ;

  // parse the input
  int sc = parse ( result , input ) ;
  if ( sc.isFailure() )
  {
    // error here ...
  }
  std::cout << "vector size " << result.size() << std::endl ;
\end{DoxyCode}
