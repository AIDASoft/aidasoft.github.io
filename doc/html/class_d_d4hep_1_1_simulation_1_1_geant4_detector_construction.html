Detailed Description

Basic implementation of the Geant4 detector construction action.

Concrete implementation of the Geant4 detector construction action. The sequences as the object's master dispatches the callbacks for the "constructGeo()" method to create a detector geometry and the "constructSensitives()" call to create the sensitive detectors and the "constructField" to each worker subclassing Geant4DetectorConstruction.

Please note: constructField() and constructSensitives() are executed in a thread-local context. See the Geant4 documentation for details: https://twiki.cern.ch/twiki/bin/view/Geant4/QuickMigrationGuideForGeant4V10
