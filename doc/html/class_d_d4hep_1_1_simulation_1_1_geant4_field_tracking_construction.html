Detailed Description

Detector construction action to perform the setup of the Geant4 tracking in magnetic fields.

Geant4FieldTrackingSetupAction.

The phase action configures the Geant4FieldTrackingSetup base class using properties and then configures the Geant4 tracking in magnetic fields.
