Detailed Description

Handle class describing a detector element.

Detector elements (class DetElement are entities which represent subdetectors or sizable parts of a subdetector. A DetElement instance has the means to provide to clients information about

  • the detector hierarchy by exposing its children.
  • its placement within the overall experiment if it represents an entire subdetector or its placement with respect to its parent if the DetElement represents a part of a subdetector.
  • information about the Readout structure if the object is instrumented and read-out. Otherwise this link is empty.
  • information about the environmental conditions etc. conditons. The access to conditions is exposed via the DetConditions interface. See DD4hep/DetConditions.h for further details.
  • alignment information.
